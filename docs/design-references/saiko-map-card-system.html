Map Card System — Spec Summary

Three Variants

Variant Size Image Use Case
2×1 Wide 2 cols × 1 row Hero (35% width) Search results, mixed grids
2×2 Featured 2 cols × 2 rows Hero (full width top) Homepage, editorial features
Title Block Flexible None Sidebars, lists, fallback

Content Hierarchy (All Variants)

  1. MAP · X PLACES — always first, always visible
  2. Title — Libre Baskerville italic
  3. Description — 1-2 lines
  4. Meta: Author + Area

Author Attribution

  • User maps: By @username
  • Saiko-curated: Curator Pick badge
  • Never show "Saiko Maps" or "Staff Pick"

Visual Differentiation

  • 2px khaki border (Place cards use 1px subtle)
  • MAP · X PLACES label is primary signal
  • Horizontal layout (2×1) vs vertical (most Place cards)
