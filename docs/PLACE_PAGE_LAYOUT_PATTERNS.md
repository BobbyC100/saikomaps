## Fallback Strategies

### Strategy 1: Promotion
When a card can't find a companion, expand it to fill the row:
```
Before:  Tips(2) + ??? (4)
After:   Tips(6) - Promoted
```

### Strategy 2: Quiet Fill
Add decorative Quiet cards to complete the row:
```
Before:  Menu(3) + ??? (3)
After:   Menu(3) + Quiet(2) + Quiet(1)
```

### Strategy 3: Companion Finding
Look for compatible cards to pair:
```
Coverage(4) → Coverage(4) + Curator(2)
Vibe(2) → Vibe(2) + Wine(3) + Quiet(1)
Tips(2) → Tips(2) + Menu(3) + Quiet(1)
```

## Constraint Enforcement

### ✅ Always True
- Every row sums to exactly 6 columns
- Span-1 cards are ONLY Quiet cards
- Max 4 Quiet units total per page
- Max 2 Quiet units per row
- Row 2 NEVER has Quiet (editorial priority)
- Row 1 is ALWAYS Hours(3) + Details(3)

### ✅ Never True
- Row sum < 6 (holes)
- Row sum > 6 (overflow)
- Functional card with span-1
- 5+ Quiet units per page
- 3+ Quiet units in one row
- Quiet card in Row 2
