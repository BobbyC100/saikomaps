## Block Priority Order

Blocks render in this order (top to bottom):

1. **Hero + Meta** (always present)
2. **Action Cards** (website, instagram, directions)
3. **Gallery** (1-4+ photos)
4. **Curator's Note** (paired with Excerpt or Vibe)
5. **Excerpt** (editorial quote)
6. **Vibe** (atmosphere tags)
7. **Hours** (business hours)
8. **Map** (styled tile + address)
9. **Call** (phone number)
10. **Tips** (bulleted list)
11. **Coverage** (editorial mentions)
12. **Best For** (description)
13. **Also On** (map cross-references)
14. **Quiet Cards** (grid fillers)
